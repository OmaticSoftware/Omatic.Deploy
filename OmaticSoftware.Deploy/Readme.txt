To install:

1) Run Powershell prompt as administrator
2) Run CustomizationInstall.ps1 script, with the following parameter:
	-InstanceName [instance name]		BB CRM instance name to use.  Defaults to "" (i.e. blank).

To see full list of CustomizationInstall.ps1 parameters, run "Get-Help .\CustomizationInstall.ps1 -Detailed"
